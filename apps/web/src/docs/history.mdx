## 履歴と謝礼

この度は、DCON 2025の茨城高専チーム「明日のDCON楽しみだね」によるプロダクト「Locker.ai」の遺失物データセットを構築するためのアンケートにご協力いただき誠にありがとうございます。

今回、1回あたり数分で終わるアンケートに一定回答えていただいた方に、🎁「モンスターエナジー パイプラインパンチ 355mL缶」をプレゼントしております！

<div className="markup_div markdown-alert markdown-alert-note">
  **条件1:** *「遺失物の画像を見ながら、説明文章を書く」* に **7件** 以上、かつ
  *「遺失物の画像を10秒だけ見て、思い出しながら説明文章を書く」* に **4件** 以上回答している
</div>

<div className="markup_div markdown-alert markdown-alert-note">
  **条件2:** *「説明文章が遺失物を正しく説明しているか投票する」* に **20件** 以上回答している
</div>

なお、両条件を満たした方は、重複して謝礼を受け取ることができます。

アンケートの記入締め切りは、📆 **2024年12月29日 (日) まで**です。

謝礼の受け渡しは下記の通り行います。

- 📆 期間: **2025年1月6日(月) から 1月10日(金) まで**
- 🏢 場所: **図書館棟1階ロビー**

謝礼を受け取るためには、下部の入力欄にメールアドレスを入力し、 _「履歴を調べる」_ ボタンを押したのち、各受け取りボタンを押してください。
