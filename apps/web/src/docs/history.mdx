## 履歴と謝礼

この度は、DCON 2025の茨城高専チーム「明日のDCON楽しみだね」によるプロダクト「Locker.ai」の遺失物データセットを構築するためのアンケートにご協力いただき誠にありがとうございます。

今回、​アンケートの協力者には、謝礼として「モンスターエナジー パイプラインパンチ 355mL」を下記の条件を満たした方にお渡ししております。

<div className="markup_div markdown-alert markdown-alert-note">
  **条件1:** 「遺失物の画像を見ながら、説明文章を書く」に **7件**
  以上、かつ「遺失物の画像を10秒だけ見て、思い出しながら説明文章を書く」に **4件** 以上回答している
</div>

<div className="markup_div markdown-alert markdown-alert-note">
  **条件2:** 「説明文章が遺失物を正しく説明しているか投票する」に **20件** 以上回答している
</div>

謝礼を受け取るためには、下部の入力欄にメールアドレスを入力し、「履歴を見る」ボタンを押したのち、各受け取りボタンを押してください。
